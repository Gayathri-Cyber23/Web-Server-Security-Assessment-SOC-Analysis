# Project Summary

This capstone project simulated a cybersecurity consulting engagement for a small business with a Linux-based WordPress e-commerce environment.

The project required the team to review the technical environment, develop a security program, implement hardening measures, conduct vulnerability analysis, monitor security events, and prepare final recommendations.

The project followed three major phases:

1. Security program development and risk assessment
2. Security implementation and penetration testing strategy
3. Vulnerability assessment and SOC-style security operations review

My role was Team Lead. I helped coordinate milestones, manage deadlines, compile deliverables, and support team communication throughout the project.
